# 02 — The 12 Domains (Districts)

This is the core content of the product: 12 life domains, each rendered as a "district." For the website, each of these maps naturally to a section, card, page, or panel.

For every domain below:
- **Label / Descriptor** — the display name and one-line "place" subtitle used in the UI.
- **Brand color** — the domain's signature hex.
- **Architecture motif** — the visual personality (from the 3D version; useful as a mood cue even for 2D).
- **Mood (tone)** — one of `core` / `surge` / `calm` / `focus`, the intended energy of the section.
- **Preview triplet** — three short lines (`Status` / `Insight` / `Signal`) written as the kind of live, contextual readout SIA would surface for that domain. These are excellent ready-made microcopy for cards.

> **Color note:** The brand colors below are the canonical per-structure colors. One discrepancy exists in the source: the **Nutrition** runtime `motifColor` is set to forest green `#34A853`, while its official brand color in the structure spec is **Warm Amber `#D97706`**. Both are recorded; pick deliberately for the website.
