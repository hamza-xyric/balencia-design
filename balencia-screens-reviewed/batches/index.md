## Batch Completion Criteria

- All selected screens reviewed one by one.
- No audit session exceeds 12 screens unless the user explicitly changes the process.
- The active batch file matches the compiled audit structure used in `batch-01.md`.
- Each reviewed screen has a `## Screen Notes` section entry with first impression, screenshot path, full rubric scores, findings, and decision.
- Findings are added to `../findings/findings-ledger.md`.
- Accepted work is copied to `../findings/accepted-improvements.md`.
- Deferred questions are copied to `../findings/deferred-decisions.md`.
