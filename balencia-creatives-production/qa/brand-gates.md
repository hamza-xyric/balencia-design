# Brand gates (hard fails)

Any **fail** requires iterate or reject regardless of rubric averages. Sourced from creative-opportunities and Balencia design system.

## Photography and people

| Gate | Fail if |
| --- | --- |
| Warm candid | Cold clinical lighting, sterile hospital vibe |
| Age and tone | Obviously stock “fitness influencer” or gym-bro aesthetic |
| Overlay | Black or heavy dark overlay on photos (warm overlays only) |
| Authenticity | Obviously fake AI hands/faces on progress or identity assets |
| Source fit | Sensitive people/body/trust assets are generated or stock-like when a real/licensed/owned or non-identifiable demo fixture is required |

## Illustration and motion

| Gate | Fail if |
| --- | --- |
| Playful restraint | Cartoonish or childish UI illustration |
| Decorative charts | Chart or graph used as decoration without data meaning |
| Motion purpose | Motion added only as wallpaper without state/product meaning |

## Product and legal

| Gate | Fail if |
| --- | --- |
| Provider marks | AI-generated Google/Apple/Spotify/WhatsApp logos (use CP-09 official assets) |
| Balencia marks | AI-generated, redrawn, distorted, or approximate Balencia logo mark/wordmark; use official files from `Balencia-Creatives-Reference/logos/` |
| Splash reveal | Generated asset contains a fake logo mark, fake wordmark, or a continuous stroke wrapping around the official logo |
| P3 overload | Large hero art on OTP, consent, or settings screens |
| Safety tone | Playful art on report/block or medication safety flows |
| Progress photos | Placeholder camera tiles while UI claims AI body analysis |
| Progress photo share | Share/export affordances on progress photos in V1 (Q18: not allowed) |
| Reports export | PDF download, data package, or external export promises on Reports (Q42: in-app only) |
| Social V1 | Global/country leaderboards, public rooms, or public proof as default visual language (Q36) |
| Knowledge Graph | Dense force-graph hero that implies freeform graph manipulation (Q43: guided insight-map) |
| Guest preview | Full fake browsable app shell when product is preview/demo placeholder (Q10) |
| Media demo | Spotify/YouTube UI that implies live sync without demo labeling (Q05) |
| Achievements | Wall of locked trophies for low-motivation default (Q39: earned/near-next focus) |
| App rating | Coercive prefilled positive or manipulative rating artwork (Q48) |

## Layout

| Gate | Fail if |
| --- | --- |
| Color saturation | All three hero colors at full saturation in one composition |
| Poster cards | Every card on a screen looks like a full poster |
| Generated text | Readable fake UI text, fake labels, or text gibberish appears inside generated art |

## Pass checklist (per asset)

- [ ] Gate review complete
- [ ] Documented in brief **Brand gate** row
- [ ] Source/license and logo integrity checked where applicable
