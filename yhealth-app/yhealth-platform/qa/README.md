# yHealth QA Feedback System

## What This Is

A crowdsourced QA system where every Xyric team member becomes a tester. Each person tests the product wearing two hats: as a real **end user** and through their **professional lens**. The result is feedback that no single QA specialist could produce alone.

## How It Works

```
1. Round Announced    → Scope defined, testers assigned, deadline set
2. Copy Template      → Each tester copies QA-Feedback-Template.md to their own file
3. Test & Fill        → 30-60 min testing, document findings in your file
4. Consolidate        → Findings compiled into recommendations for team meeting
```

When you create your feedback file, name it:

```

```

Place it in the current round folder: `rounds/R001-mvp-pre-launch/`
