# Balencia / SIA — Website Handoff Package

**Balencia** is the brand for **SIA**, an AI "life operating system" that connects every dimension of a person's life — fitness, finance, relationships, sleep, career, and more — into one intelligent daily plan. The original product was prototyped as a **3D cinematic city** (a central "SIA Tower" surrounded by 11 district buildings, one per life domain) that users scrolled through as a 17-scene fly-through.

We are now **moving away from 3D modelling** and rebuilding this as a **conventional website** that presents the same information. This folder is a self-contained brief: it captures the product concept, all the copy, the visual identity, the per-domain data, the story arc, and the in-app product mockup — everything a new website effort needs **without opening the 3D source**. It is descriptive only; it does not prescribe how to build the new site.

> All values here were taken verbatim from the existing source files (see `08-source-pointers.md` for exactly where each lives). Where the source contains a discrepancy, it is flagged inline.
