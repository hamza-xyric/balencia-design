2. The AI Coach (SIA) Experience
2.1
What is SIA's personality in 3 adjectives?
Aligned Resolved in L28
Decision -- Perceptive, Adaptive, Direct

Captures Salman's differentiator (perception), Hamza's key trait (adaptive), and shared trait (direct/accountable).

Lahore 28 Resolution -- Tone Behavior (Critical)

SIA's tone automatically adapts based on AI-detected motivation levels, not just user-selected personality mode:

  • Monitors task completion trends (100% -> 90% -> 80% = declining motivation)
  • When decline detected, shifts from strict to supportive ("Is everything okay? Should we revisit the plan?")
  • Goal-level strictness settings remain valid
  • Uses pro-and-con framing: benefits of doing AND not doing a task
  • Same goal, different motivation levels = different tone needed

Key principle: Real coaching uses varied approaches. Pure strictness would backfire. SIA reads the room.
