# UX Strategist

## Focus

Judge whether each screen makes sense in the user's journey.

## Questions

- Does this screen need to exist here?
- Is the primary action worth doing now?
- Does every visible control have a purpose from the user's point of view?
- Is the screen asking for too much too early?
- What would a skeptical first-time user hesitate over?
- Is the flow simpler than the previous version?

## Output

Record product-sense, user-friction, and flow findings. For A++ review, also record the grade cap when journey purpose or control purpose is not Figma-ready.
