# Balencia Branding

## Brand Name
- Balencia

## Logo Concept
- **Type style:** `Monda Regular`
- **Concept note:** An intentional visual tension created by shifting geometric weights to represent the human state before coaching begins.

## Core Brand Colors

### 1) Burnt Orange
- **Hex:** `#FF5E00`
- **Role:** Primary action color (CTA buttons, high-attention highlights)

### 2) Forest Green
- **Hex:** `#34A853`
- **Role:** Progress/recovery/success states and supportive highlights

### 3) Royal Purple
- **Hex:** `#7F24FF`
- **Role:** Intelligence/AI/coaching accents, secondary emphasis

## Typography
- **Primary Typeface:** `Monda`
- **Recommended Weights:** `400`, `700`
- **Usage:**
  - Headings: Monda 700
  - Body: Monda 400
  - UI labels/buttons: Monda 700

## Brand Tone (Visual)
- Bold, energetic, and modern
- Human-centered but data-intelligent
- High contrast with clean geometric expression

## Suggested UI Application
- **Primary Button:** Background `#FF5E00`, text white
- **Secondary Accent/Hover:** `#7F24FF`
- **Success/Recovery Indicators:** `#34A853`
- **Gradient Direction:** Blend among orange, purple, and green for hero accents and data visualization highlights

## Accessibility Notes
- Maintain minimum WCAG contrast ratios for text on brand colors
- Prefer white text on orange/purple backgrounds
- For green surfaces, use dark text when background is light and white text when background is deep
